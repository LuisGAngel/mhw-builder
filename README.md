# MHW Builder

A Monster Hunter World armor/build planner built with Angular. Plan your builds, compare stats, and optimize your loadout.

**Live App:** https://luisgangel.github.io/mhw-builder/

## What's New

This fork extends the original project with multiple quality-of-life features:

- **Build Solver** — An armor and decoration optimizer that automatically finds optimal gear combinations for your desired skills.
- More QoL improvements coming soon.

## Development Server

Run `npm start` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

## Production Build

```bash
npm run build
```

## Deploy to GitHub Pages

```bash
npm run deploy
```

Builds with `--base-href /mhw-builder/` and deploys the `dist/` folder to the `gh-pages` branch.
